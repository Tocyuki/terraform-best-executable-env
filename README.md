# Terraform Best Executable Environment
The Best of Executable Environment of Terraform for you!!

## Requirement
- docker-compose
- make

## Usage
### terraform init
```
$ make init
```

### terraform plan
```
$ make plan
```

### terraform apply
```
$ make apply
```

### terraform destroy
```
$ make destroy
```

### terraform fmt check & validate
```
$ make check
```
